Git Started

Apr 22nd, 2013

I recently replaced my Mac. New hardware and new OS.

Now was the time to get it configured as an awesome programming machine. I plan to list all what and how in the next post.

At this time I going to start with how did I get this started.

  1. Set up Git. The first time I ran through this, I realized that I hadn’t set up by ssh keys on this computer. This caused a bunch of permission error when I tried to deploy.
    https://help.github.com/articles/generating-ssh-keys
  2. Start with Octopress.
    http://octopress.org/
  3. How to set some better configuration.
    http://www.moncefbelyamani.com/how-to-install-and-configure-octopress-on-a-mac/
  4. Create your repository in Git. I forgot that step the first time also.
    https://github.com/new
  5. Ready to deploy.
    http://octopress.org/docs/deploying/github/
  6. Wait. After you deploy, GitHub still needs to get the pages built. The page says that they will send you an email when ready, but I never got one.
